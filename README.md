# StayVista - Rental Listing Platform 🏡

A full-stack rental listing web application inspired by Airbnb, built using Node.js, Express.js, MongoDB, EJS, Passport.js, Cloudinary, and Mapbox.

## Features

- User Authentication (Login/Signup)
- Create, Edit & Delete Listings
- Image Upload with Cloudinary
- Search Listings
- Ratings & Reviews
- Interactive Maps with Mapbox
- Responsive UI

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Passport.js
- Cloudinary
- Mapbox
- Bootstrap

## Installation

```bash
git clone <repository-url>
cd StayVista---Rental-Listing-Platform
npm install
```

Create a `.env` file with:

```env
ATLASDB_URL=your_database_url
MAP_TOKEN=your_mapbox_token
CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_key
CLOUD_API_SECRET=your_cloudinary_secret
SECRET=your_secret
```

Run the project:

```bash
npm start
```
